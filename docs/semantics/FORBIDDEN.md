# FORBIDDEN

agentic-flows explicitly refuses to support:

- Implicit memory across runs.
- Best-effort execution or partial success semantics.
- Opaque chain-of-thought or hidden reasoning.
- Silent retries or hidden backoff.
- Probabilistic verification or non-reproducible verification.
- Undeclared side effects or undeclared external calls.
- Untracked artifacts or evidence.
- Mutable artifacts.
- Non-deterministic execution paths.
- Verification without evidence.
